# Light Falling

A lightweight Minecraft utility mod for enhancing gameplay with quality-of-life features.

## Features

- **Automation** - Auto mining, farming, and tool switching
- **Visualization** - Redstone signal indicators, customizable HUD, 3D overlays
- **Customization** - Adjustable FOV, brightness, and visual settings
- **Survive** - Anti-damage features, enhanced controls

## Building

```bash
./gradlew build
```
